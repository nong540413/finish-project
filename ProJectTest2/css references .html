css refernces

transform-origin: top center;: คุณสมบัติ CSS นี้กำหนดจุดกำเนิดที่สำหรับการทำแปลง ในที่นี้กำหนดจุดกำเนิดที่ด้านบนตรงกลางขององค์ประกอบ โดยเมื่อนำมาใช้ร่วมกับการทำแปลง เช่นการหมุนหรือปรับขนาด จุดกำเนิดนี้จะระบุจุดที่แปลงเกิดขึ้น

transform: scale3d(1, 0, 1): คุณสมบัติ CSS นี้นำมาใช้กับการปรับขนาดแบบ 3 มิติ โดยค่า 1, 0, และ 1 แทนปัจจัยการปรับขนาดตามแกน x, y, และ z ตามลำดับ ในที่นี้, องค์ประกอบจะถูกปรับขนาดเพื่อให้สูงเป็น 0, ทำให้หายไปในทิศทางแนวตั้ง

cursor: pointer: คุณสมบัติ CSS นี้เปลี่ยนรูปลักษณะของเคอร์เซอร์เมื่อชี้ไปที่องค์ประกอบ การตั้งค่าเป็น pointer บ่งชี้ว่าองค์ประกอบนี้เป็นองค์ประกอบที่สามารถคลิกได้, มักใช้สำหรับลิงก์หรือปุ่ม

video: อ้างถึงองค์ประกอบ HTML video ใช้ในการฝังเนื้อหาวิดีโอในหน้าเว็บ

  • autoplay: คำสั่งทำให้สื่อ (เช่นวิดีโอหรือเสียง) เริ่มเล่นโดยอัตโนมัติเมื่อหน้าเว็บโหลดเสร็จ.

    loop: คุณสมบัตินี้ทำให้สื่อเล่นวนลูปโดยไม่ต้องมีการกระทำจากผู้ใช้หลังจากที่ถึงจุดสิ้นสุดของสื่อ.

    muted: การเปิดคุณสมบัตินี้ทำให้เสียงของสื่อถูกปิด, ดังนั้นสื่อจะเล่นโดยไม่มีเสียง. นี่เป็นประโยชน์ในกรณีที่คุณต้องการวิดีโอทำงานอัตโนมัติและวนลูป, แต่ไม่ต้องการเสียงที่รบกวน

svg: อ้างถึงองค์ประกอบ HTML svg ใน SVG (Scalable Vector Graphics) เป็นรูปแบบรูปภาพเวกเตอร์ที่ใช้กำหนดกราฟิกสำหรับเว็บ เพื่อให้มีความยืดหยุ่นและสามารถทำงานได้ในหลายขนาด

  • g (Group Element): ใช้เพื่อกลุ่มหลาย ๆ องค์ประกอบเข้าด้วยกันในกลุ่มเดียว. มีไว้เพื่อการจัดระเบียบและการจัดการขององค์ประกอบ

    path (Path Element): ใช้สร้างเส้นทาง (path) หรือเส้นประทาน (outline) โดยใช้คำสั่งต่าง ๆ เช่นการเคลือบ (move), เส้นตรง (line), และการวาดวงกลม (arc) ซึ่งจะช่วยในการสร้างรูปร่างที่ต้องการ
